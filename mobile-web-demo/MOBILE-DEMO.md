# 🚀 手机端指挥开发快速迭代演示

## 项目目标

通过手机端指挥AI进行web开发，并在手机浏览器实时查看效果，实现快速迭代优化。

## 演示内容

### 1. 基础页面 (index.html)
- 主页面展示核心功能
- 颜色选择器
- 计数器

### 2. QR Code页面 (QR-Code.html)
- 二维码样式展示
- 部署流程说明

### 3. 本地服务器 (serve.sh)
- 启动本地服务器
- 手机浏览器访问
- 实时更新功能

## 手机端指挥流程

### 发送指令示例：
```
"修改主页标题为'手机端指挥开发平台'"
```

### AI响应流程：
1. AI接收指令
2. 修改 index.html 文件
3. 提交代码变更
4. 服务器重新加载
5. 手机刷新查看变化

### 实际效果：
- 你在手机上发送指令
- 我在后台修改代码
- 你的手机浏览器即时看到更新

## 手机访问方式

### 本地访问 (同一局域网):
```bash
./serve.sh
```
然后在手机浏览器访问：http://[电脑IP]:8000

### GitHub Pages (上线后):
访问生成的 GitHub Pages URL
