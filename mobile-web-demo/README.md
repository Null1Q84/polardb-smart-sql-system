# 手机端指挥开发演示

这是一个通过元宝手机端指挥AI进行web项目开发，并在手机浏览器快速体验的演示项目。

## 特性

- 🚀 **实时部署**：GitHub Actions自动部署到GitHub Pages
- 📱 **手机体验**：手机浏览器访问，随时查看最新版本
- 🤖 **AI协助**：通过元宝手机端指挥AI进行代码修改
- 🔁 **快速迭代**：快速反馈循环，实时优化改进

## 项目结构

```
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # JavaScript脚本
├── README.md           # 项目说明
├── .github/workflows/  # GitHub Actions配置
│   └── deploy.yml      # 部署配置文件
```

## 使用方法

1. **手机端指挥**：
   - 在元宝手机端发送指令（如："修改标题颜色"、"添加新功能"）
   - AI会自动修改代码并推送更新

2. **访问体验**：
   - 手机浏览器访问部署后的URL
   - 扫码访问QR Code
   - 查看最新版本功能

3. **快速迭代**：
   - AI修改代码后立即部署
   - 手机刷新即可查看更新

## 部署流程

每次代码推送到main分支后，GitHub Actions会自动：
1. 部署到GitHub Pages
2. 生成QR Code图片
3. 创建artifact供下载

## 开发流程

1. 手机端发送开发指令
2. AI修改代码文件
3. AI提交代码到GitHub
4. GitHub Actions自动部署
5. 手机访问体验新功能

## 手机端建议

- 使用手机浏览器访问，支持现代浏览器特性
- 可以扫描QR Code快速访问
- 通过元宝发送自然语言指令控制开发
