📡 API调用演示

这是AI大模型调用的演示架构。你可以通过前端调用AI API来实现真正的AI功能。

API调用流程

  1. 前端发送请求到AI API代理服务器
  2. AI模型处理请求并返回结果
  3. 前端接收结果并实时展示
  4. 用户可以与AI进行真正的对话

💬 AI对话模拟

🤖: 你好！我是AI助手。你可以问我关于网页开发的问题，比如："帮我创建一个按钮组件"、"设计一个登录表单"、"实现一个图表功能"。

如何实现真正的AI调用

方案一：前端直接调用API（需要API密钥）

async function callRealAI(prompt) {
    const response = await fetch('https://api.openai.com/v1/chat/completions', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': 'Bearer YOUR_API_KEY'
        },
        body: JSON.stringify({
            model: 'gpt-4',
            messages: [{ role: 'user', content: prompt }],
            max_tokens: 500
        })
    });
    return response.json();
}

方案二：后端代理调用（更安全）

// 前端调用你的服务器
async function callAIProxy(prompt) {
    const response = await fetch('https://your-server.com/api/ai-proxy', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ prompt })
    });
    return response.json();
}

方案三：实时AI代码生成

// AI根据描述生成HTML/CSS代码
async function generateCode(description) {
    const aiResponse = await callRealAI(`生成HTML/CSS代码: ${description}`);
    const generatedCode = aiResponse.choices[0].message.content;
    
    // 将生成的代码实时插入页面
    document.getElementById('generated-code').innerHTML = generatedCode;
}
